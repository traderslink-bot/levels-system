# TradersLink AI Read Archive

`archive.json` contains the complete TradersLink AI Read payloads that were
available in the local live-watchlist audit archive at export time.

The file is intended to give ChatGPT and other repository tools direct,
credential-free access to prior reads. Records are ordered by `generatedAt`
and include the ticker, generation ID, timestamps, and original AI Read payload.

This snapshot contains 45 reads across 45 tickers. It is a point-in-time export,
not a complete version history: reads that had already expired from or been
overwritten in the source archive could not be recovered.
